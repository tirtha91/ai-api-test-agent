# AI API Test Generator Agent

## 🚀 Overview
AI-powered agent that converts OpenAPI specs into executable Pytest scripts

## 🧠 Features
- Auto test generation using LLM
- Positive & negative cases
- AI-based code refinement
- Self-improving review loop
- Handles path params & data dependencies

## ⚙️ Tech Stack
Python, OpenAI, Pytest

## ⚙️ Setup
pip install -r requirements.txt

export OPENAI_API_KEY=your_key

## ▶️ Run
python main.py
pytest tests/
